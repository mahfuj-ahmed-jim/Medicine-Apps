# Medicine-Apps

A medicine reminder app that fulfills the userâ€™s medication needs in one place. Users can add and track unlimited amounts of medicine with their proper details. This application also helps users to track their medical history.

Play Store Link: https://play.google.com/store/apps/details?id=com.ai.medicinereminder&pli=1


## Features

- Add medicine with proper information 
- Show everyday's remaining medicine
- Warning for missing medicine
- Alarm for medicine time
- All medical history including medicine


## Tools

- Android Studio
- Java
- Firebase
- Room Database
